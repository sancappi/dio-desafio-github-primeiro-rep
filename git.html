Introdução ao Git

Git pode ser entendido como um software de versionamento de código. Sua forma de navegação inicial foi por meio da linha de comando. Comandos básicos para serem usados no terminal linux: cd, ls, mkdir, rm, clear (Ctrl+l)...

O funcionamento do git se dá através de quatro tópicos:

SHA1 - Secure Hash Algorithm (Algoritmo de criptação), serve como uma forma curta de identificar um estado do arquivo.

Objetos internos do git - São os responsáveis pelo versionamento de código.

  • Blobs
  • Trees
  • Commits

O git é considerado um sistema distribuído pois permite ter uma versão do arquivo no repositório local e outra no repositório remoto.

Desde 2021 o Git não possui mais suporte para senhas, portanto, o token de acesso pessoal passou a ser prioridade na hora da autenticação do usuário. Informações para criá-lo e configurá-lo clique aqui!
